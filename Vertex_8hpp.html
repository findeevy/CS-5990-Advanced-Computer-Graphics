Detailed Description

Defines the Vertex struct used in Vulkan rendering pipelines.

The Vertex struct encapsulates all per-vertex attributes required for rendering — position, color, and texture coordinates — and provides helper functions for describing these attributes to Vulkan.

This struct is designed to align precisely with shader input layouts. It provides static functions to retrieve binding and attribute descriptions that are used during Vulkan pipeline creation.

Warning
If you modify field order or types, update both getBindingDescription() and getAttributeDescriptions() to maintain compatibility with your shaders.
Note
Shader input locations must match the attribute descriptions.
