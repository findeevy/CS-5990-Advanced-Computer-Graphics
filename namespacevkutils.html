Creates a Vulkan image view for a given image resource.

An image view describes how Vulkan should access an image, specifying the format, subresource range, and mip levels. This function encapsulates the Vulkan setup boilerplate into a single RAII-based helper.

Parameters
deviceLogical Vulkan device used to create the view.
imageTarget Vulkan image.
formatImage format (e.g., vk::Format::eR8G8B8A8Srgb).
aspectFlagsSpecifies which aspects (color, depth, stencil) are included.
mipLevelsNumber of mipmap levels in the image. Defaults to 1.
Returns
A vk::raii::ImageView wrapping the created image view.
Exceptions
vk::SystemErrorIf Vulkan fails to create the image view.
Note
Typically used for texture or swap chain images before rendering.

std::vector< char > vkutils::readFile ( const std::string &  filename)

Reads a binary file (e.g., SPIR-V shader) into a vector of bytes.

Opens a file in binary mode, seeks to the end to determine size, and reads the contents into a std::vector<char>.

Parameters
filenamePath to the file to read.
Returns
std::vector<char> containing the complete binary contents.
Exceptions
std::runtime_errorIf the file cannot be opened or read.
Note
Commonly used to load SPIR-V shader bytecode for Vulkan pipeline creation.
auto vertShaderCode = vkutils::readFile("shaders/vert.spv");
auto fragShaderCode = vkutils::readFile("shaders/frag.spv");
