Detailed Description

Provides a 'std::hash' specialization for the Vertex struct.

This header defines a custom hash functor that enables Vertex objects to be used as keys in unordered associative containers such as 'std::unordered_set‘ and 'std::unordered_map’.

The hash function combines position, color, and texture coordinate data to generate a unique hash value consistent with Vertex::operator==.

Note
If you modify the layout or members of Vertex, this hash function must be updated to maintain consistency.
