Detailed Description

Aggregated statistics for a given profiling zone.

ZoneStats collects incremental statistics for a zone across multiple frames. It is intentionally simple: it accumulates total time, tracks the maximum observed sample, and counts samples so callers can compute averages.

Note
All times are expressed in milliseconds.
