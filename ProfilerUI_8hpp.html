Detailed Description

Console-based ASCII UI for visualizing ChronoProfiler results.

This header declares a minimal, dependency-free ASCII profiler UI that:

  • maintains a rolling history of frames,
  • prints a per-frame breakdown as ASCII bars,
  • accumulates aggregated statistics (avg/max/count) per zone,
  • exposes a no-op implementation when the profiler is disabled so call sites can remain free of '#ifdef' guards.

The real implementation is compiled only when the preprocessor symbol 'PROFILER' is defined (e.g. pass '-DPROFILER' or 'make PROFILING=1').

Note
The UI is intentionally lightweight: it does not depend on ncurses, ImGui, or other GUI libraries. It is intended for quick in-terminal inspection during development.
