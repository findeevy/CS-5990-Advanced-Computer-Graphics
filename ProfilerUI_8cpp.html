Detailed Description

Implementation of the ProfilerUI class for console-based visualization of ChronoProfiler CPU profiling events.

This file contains the implementation of ProfilerUI methods, responsible for:

  • Storing frame history
  • Aggregating zone statistics
  • Rendering a simple ASCII-based profiler view in the terminal

This implementation only exists when compiled with '-DPROFILER'. Otherwise, the header provides a zero-overhead stub implementation.
